# ChatWork-to-GoogleSheets2

## 概要

- [ChatWork-to-GoogleSheets](https://github.com/amada717/ChatWork-to-GoogleSheets) から、**メッセージエンジニアリング** を重視したものです
- ある意味、自由度をなくし、メッセージのテンプレート入力によって「縛り」をつけたものです。
- カテゴリーやタグについても、こちらで用意した項目を選択してもらうようなことになります。
- つまり、メッセージ作成者には、カテゴリーやタグ作成に考察する力をそそがないようにしてもらい、メッセージ本文の内容作成に集中してもらうのが目的となり、よりシステマチックにバックエンドでは処理を行うというものです。
- また、今回はWEBアプリ作成まではせず、シートに保存することにとどめ、**テンプレート作成（メッセージエンジニアリング）に集中する**ものです。
